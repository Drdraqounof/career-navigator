Find Me

Empowering students, graduates, and career changers with personalized guidance based on real job market trends

Project Overview

🎯 What It Does

A web app that helps users figure out which jobs fit them, what skills they need, and what steps to take next. Provides personalized guidance based on real job market trends.

🏗️ Development Approach

Built using Agile methodology with 3 sprints over 6 weeks. Core features first, then advanced functionality. Mock data initially, real APIs later.

📊 Smart Insights

Helps improve resumes, identifies skills gaps, and creates actionable career plans. Adapts recommendations as the job market evolves.

Who It's For

🎓 High school seniors planning college or careers
👨‍🎓 Recent graduates entering the job market
🔄 Career changers looking for new opportunities
🔍 Anyone wanting smarter job search methods

Problems We Solve

Resume Struggles: Don't know how to tailor resumes for specific jobs
Skills Gaps: Unclear about what skills they're missing
Learning Direction: Unsure what to learn next in their career
Market Trends: Don't understand current job market dynamics

Key Features

🛤️ Onboarding Flow

Users answer personalized questions to receive tailored career advice and recommendations.

🔍 Explore Flow

Search and filter careers with detailed information about requirements, salaries, and growth potential.

💾 Save & Compare

Save interesting jobs or career paths and compare multiple options side by side.

📋 My Plan Page

Comprehensive summary of skills gaps and suggested actionable next steps.

📄 Export Feature

Download personalized career plans as PDF or Markdown files for offline reference.

Development Timeline

Sprint 1: Foundation (Weeks 1-2)

Week 1: Define problem, analyze competitors, create user personas, plan MVP

Week 2: Create wireframes, set up Trello backlog, scaffold project repository

Sprint 2: Core Setup (Week 3)

Build onboarding flow, set up application state management, implement mock APIs, establish design system

Sprint 3: Main Features (Weeks 4-5)

Week 4: Build Explore flow, Save & Compare functionality, data persistence

Week 5: Implement My Plan page, export features, conduct usability testing

Final Phase (Week 6)

Complete documentation, create training materials, prepare demo, showcase final project
