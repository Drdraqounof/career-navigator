# Career Navigator

An interactive career quiz app built with HTML, CSS, and JavaScript.

## 🚀 Features
- Career quiz with multiple choice questions
- Career group matching
- Results with match percentages

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Alpine.js

## Why Was This Made?
This was orginally made for an assignment to test if we would be able to use what we,
learned in vs code and use it based of a case study prompt The prompt focused on the rise of AI in the job market and the challenges faced by new job seekers who struggle to compete due to limited experience and resume-related issues. From their we made wireframes and Visonboards to plan out our ideas on the user should feel and what the user should learn form and take away from the website/App. Then to back it up I did indepth documentation to make sure things aligned with not only the propmt given but base on real world problems related to the propmt
